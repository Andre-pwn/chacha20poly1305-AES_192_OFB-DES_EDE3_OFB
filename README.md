### chacha20poly1305-AES_192_OFB-DES_EDE3_OFB

## Installation

No installation is required

## Modules

You should install python3, os, time, chacha20poly1305 and Cryptodome

## Usage
Change the file path and test if the cyphers works

```bash
python3 main.py
```
